#addStore

v1.0.0

A pure JavaScript E-Commerce / Shopping Cart jQuery plugin.

---

A pure JavaScript E-Commerce / Shopping Cart jQuery plugin. This plugin uses Stripe to process payments and PHP to server the charges to stripe, but the cart itself is written in JavaScript/jQuery.
